# Memory Game — Vue.js

Projet EFREI • B2-DEV • 2025-2026

## Présentation

Ce projet est une application Web réalisée en **Vue.js (Composition API)** permettant de jouer au célèbre **jeu du Memory**.
Le joueur doit retrouver toutes les paires de cartes dans une grille dont la taille dépend du niveau de difficulté choisi.
L'application enregistre les parties, permet de consulter les scores, de trier les résultats et de gérer les pseudonymes.

---

## Fonctionnalités

### Gameplay

* Grilles disponibles : **4×4**, **5×5**, **6×6**
* Positionnement **aléatoire** des cartes à chaque partie
* Retourner deux cartes puis vérifier la paire
* Victoire dès que toutes les cartes sont découvertes
* Saisie d’un **pseudo** à la fin de la partie
* Possibilité de recommencer plusieurs parties

---

### Gestion des parties

* Enregistrement automatique des parties
* Affichage du **pseudo**, **niveau de difficulté**, **temps**, **nombre d’essais**
* Trie des résultats (selon votre implémentation : temps, essais, difficulté, etc.)
* Modification **du pseudo uniquement**
* Suppression d’une partie

---

### Interface & ergonomie

* Interface claire et agréable à utiliser
* Design libre mais soigné
* Thème des cartes libre
* Navigation intégrée
* Application full Vue.js (Composition API)
